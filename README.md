# soshal
名前は仮

## 開発手順

```
cp -f .env.example .env
pnpm dev
```

http://localhost:3000 を開く

### 連合の動作確認が必要な場合

```
cp -f .env.docker .env
./scripts/setup-for-docker.sh
docker compose up -d --build
```

https://soshal.localhost を開く
