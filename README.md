# soshal

名前は仮

## 開発手順

pnpm, mkcert, docker が必要

```
./scripts/setup-for-docker.sh
cp -f .env.example .env
pnpm i
pnpm dev
```

https://soshal.localhost  
https://misskey.localhost を開く
